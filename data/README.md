# Data Documentation

This project uses publicly available housing and spatial data.  
Raw data files are not included due to licensing and size considerations.

## Zillow Home Value Index (ZHVI)
- Monthly county-level home value estimates
- Covers single-family residences and condominium/co-op properties
- Source: Zillow Research
- https://www.zillow.com/research/data/

## County Boundary Shapefiles
- U.S. Census Bureau county boundaries
- Accessed via the `tigris` R package
